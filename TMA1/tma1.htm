Part 1: XML Resume

This first project required us to create an xml version of our resume. There were 3 parts required:

  • resume.xml - the resume itself as an xml
  • resume.xsd - the schema of the resume
  • resume.xsl - style sheet to display resume

For this project, the first step was to create the schema that logically represented what you would expect to see in a resume. After the schema was done, I had to format my own resume to fit this new schema. Lastly, I created the style sheet to render the XML resume in a way that was easily readable and asthetic.

Part 2: Educational Webpage

The second project is a web application that can help someone learn web technologies that we have learned in the course.

For this project, I had to decide on topics from Unit 1, 2 and 3 that I thought were fundamental, and figure out how to explain them in a way that was logical and made sense. In addition, after writing the tutorials, I had to come up with a quiz questions that tested the material well and showed that the user has understood the content correctly. To make the quizes easily editable and make it easy to add new ones, I decided to store the tests as XML files, and wrote javascript code that would automatically create the html content for the test from the XML file. This makes it easy to edit existing tests or create new ones since only the XML files need to be modified.
