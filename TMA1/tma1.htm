After the schema was done, I had to format my own resume to fit this new schema. Lastly, I created the style sheet to render the XML resume in a way that was easily readable and asthetic.

Part 2: Educational Webpage

The second project is a web application that can help someone learn web technologies that we have learned in the course.

For this project, I had to decide on topics from Unit 1, 2 and 3 that I thought were fundamental, and figure out how to explain them in a way that was logical and made sense. In addition, after writing the tutorials, I had to come up with a quiz questions that tested the material well and showed that the user has understood the content correctly.

Each unit's content is written in a seperate file, and the project uses AJAX to quickly switch between the selected unit without refreshing the whole page. The entrypoint for this project is /part2/index.html.

To make the quizes easily editable and make it easy to add new ones, I decided to store the tests as XML files with the following format:

<quiz>
 <question>
  <description>Question text here</description>
  <options>
   <option>option 1</option>
   <option correct="true">option 2</option>
   <option>option 3</option>
   <option>option 4</option>
  </options>
 </question>

 <question>
  ...

</quiz>

I then wrote javascript code that would automatically create the html content for the test from the XML file, by adding a custom quiz tag.

<quiz name="unitname" />

The name attribute defines the name of the quiz. For example, the code above would look for a file called unitname.xml and create a quiz from it. This makes it easy to edit existing tests or create new ones since only the XML files need to be modified.

Part 4: Utility Tools

For this project, we had to create a web application that has a variety of useful utility tools and calculators. The application will have the following 3 tools:

  • Measurements converters

    For this tool I created measurement converters for weight, length, area and volume, and used the following formulas.

    1 kilogram = 2.20462 pounds

    1 meter = 3.28084 feet

    1 square meter = 10.7639 square feet

    1 cubic meter = 35.3147 cubic feet

    • These formulas were taken from the Google unit converter. You can read more about it here.

  • A mortgage calculator

    This tool calculates the monthly mortgage payment for a loan given the amount, interest and period. The formula I used was taken from thebalance.com, from this article.

    a = starting loan amount
    n = number of payments
    i = periodic interest rate

    D = {[(1 + i) ^n] - 1} / [i(1 + i)^n]

    monthly payments = a/D

  • Color picker

    The last topic was a utility of our choosing. I decided to make a color picker that has a selector and displays the HEX and RGB values of the colors chosen. This is a very useful utility when creating stylesheets since you can find out the hex code of the color you want for your web applications and copy it directly into your style sheet, instead of being limited to the pre-defined colors.

    • I read this article to learn about rgb and hex color representations and how to convert between them.

    I used a canvas to create the color picker using gradients. The getImageData function allowed me to get the RGB values, then I used .toString(16) to convert each value to hexadecimal.
