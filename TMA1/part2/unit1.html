Unit 1 - Intro to the Web

This unit will teach you the most basic buiding blocks of any website or web application. That is HTML.

1.1 Basics of HTML

HTML is the basic building block of most websites. HTML stands for Hypertext Markup Language, and is what defines the content and structure of a webpage.

To get started, open your text editor and create a new file. Call it index.html.

Paste the following code into your textfile, and save it.

<!DOCTYPE html>
<html>
 <body>
  <h1>A heading</h1>
  <p>Some text here</p>
 </body>
</html>

The first line <!DOCTYPE html> specifies that this is an HTML document.

HTML syntax is made up of tags, that are enclosed in <>

Most tags need to be closed by having a / in the closing tag. As you can see from the example, the <body> tag is closed by </body>. That is the same for both h1 and p.

This is a very basic HTML file. If you open it in a web browser you should see the following:

A heading

Some text here

Enclosed in the html tags is all the HTML content for this page. The body specifies the content that is visible on the webpage.

h1 is a header, and p specifies a paragraph. There are many elements in HTML. The next section covers some of them.

Another important HTML element is div, which lets you define different sections. Generally, different portions of the webpage are split into different divs. The following example shows two different divs that correspond to different sections. We also show one more element, hr, which defines a horizontal line / break.

<!DOCTYPE html>
<html>
 <body>
  <div>
   <h1>Section 1</h1>
   <p>This is the first section</p>
  </div>
  <hr/>
  <div>
   <h1>Section 2</h1>
   <p>This is the second section</p>
  </div>
 </body>
</html>

HTML allows you to have lists. There are two types: ordered lists (ol) and unordered lists (ul). Inside the list, you specify list items using the tag li.

<html>
 <body>
  <ol>

   <li>Second item</li>
   <li>Third item</li>
  </ol>
  <ul>
   <li>First item</li>
   <li>Second item</li>
   <li>Third item</li>
  </ul>
 </body>

1.3 Input Types

HTML also allows having input. To have input, it is specified inside a form. You can create different forms for different input groups and have as many inputs in a form as you like.

To specify the type of input, you must specify the property type. These are some examples of types available in HTML.

  • text: Allows a regular text input
  • number: Only allows numbers
  • checkbox: Adds a checkbox that can be checked or unchecked
  • radio: Adds a radio button which only allows one selection at a time
  • date: Allows specifying a date input
  • reset: Button that clears the form

You can also add descriptions to your inputs by wrapping the input in a label tag.

Here is an example that uses all of those input types:

<!DOCTYPE html>
<html>
 <body>
  <form>
   <label>Text<input type="text"/></label>
   <label>Number<input type="number"/></label>
   <label>Checkbox<input type="checkbox"/></label>
   <label>Radio 1<input name="example" type="radio"/></label>
   <label>Radio 2<input name="example" type="radio"/></label>
   <label>Date<input type="date"/></label>
   <label>Reset<input type="reset"/></label>
  </form>
 </body>
</html>

The radio button input uses the property name to link the two radio buttons together. As you can see, only one of them can be selected at a time, as only one value should be present.
